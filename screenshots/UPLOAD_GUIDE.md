# 📸 Screenshots Upload Guide

## Tên file cần upload vào thư mục này:

### Authentication Screens
- `login.png` - Màn hình đăng nhập
- `register.png` - Màn hình đăng ký
- `verification.png` - Màn hình nhập mã OTP
- `forgot_password.png` - Màn hình quên mật khẩu
- `create_password.png` - Màn hình tạo mật khẩu mới

### Main Task Screens
- `home_page.png` - Màn hình tất cả task (All)
- `completed_page.png` - Màn hình task đã hoàn thành
- `uncompleted_page.png` - Màn hình task chưa hoàn thành
- `deleted_page.png` - Màn hình task đã xóa

### User Profile & Navigation
- `drawer_menu.png` - Drawer navigation menu
- `profile_page.png` - Màn hình hồ sơ người dùng
- `welcome_screen.png` - Màn hình chào mừng

## 📝 Hướng dẫn upload:

1. **Chụp/Chuẩn bị ảnh theo danh sách trên**
   - Kích thước tối ưu: 800x1600 pixels (hoặc 400x800 pixels)
   - Format: PNG hoặc JPG

2. **Copy ảnh vào thư mục này**
   ```bash
   # Đặt các file ảnh vào thư mục: screenshots/
   ```

3. **Commit và push lên GitHub**
   ```bash
   git add screenshots/
   git commit -m "Add app screenshots"
   git push origin main
   ```

## ✅ Kết quả

Sau khi push, README.md sẽ hiển thị tất cả các ảnh screenshots theo layout đẹp!

---

**Lưu ý:** Nếu muốn tối ưu hóa ảnh trước upload:
- Nặn giảm kích thước hoặc dùng tool compressor
- Đảm bảo hình ảnh sáng, dễ nhìn
- Có thể cắt ảnh để hiển thị phần UI chính
